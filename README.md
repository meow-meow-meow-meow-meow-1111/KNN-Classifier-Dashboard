🚀 An interactive **K-Nearest Neighbors (K-NN) Classifier Dashboard** built with **Streamlit**, designed for visualizing classification, tuning parameters, and making real-time predictions.

[![Live App](https://img.shields.io/badge/🚀%20Launch-App-blue?style=for-the-badge)](https://knnclassifierbychinmay.streamlit.app/)

---

## 📌 Features

* 🎯 **Real-time Predictions**

  * Input Age & Salary to predict purchase behavior

* ⚙️ **Interactive Model Controls**

  * Adjust number of neighbors (K)
  * Modify test/train split ratio

* 📊 **Performance Metrics**

  * Accuracy Score
  * Confusion Matrix visualization

* 📉 **Decision Boundary Visualization**

  * Dynamic contour plot using Plotly
  * Clear class separation view

* 📁 **Dataset Viewer**

  * Toggle to inspect raw dataset

---

## 🧠 Machine Learning Details

* Algorithm: **K-Nearest Neighbors (K-NN)**
* Distance Metric: **Euclidean (Minkowski p=2)**
* Feature Scaling: **StandardScaler**
* Dataset: **Social Network Ads**

---

## 🛠️ Tech Stack

* **Frontend/UI**: Streamlit
* **Data Handling**: Pandas, NumPy
* **Visualization**: Plotly
* **ML Model**: Scikit-learn

## ⚡ Installation & Run Locally

```bash

git clone https://github.com/your-username/knn-dashboard.git

# Navigate to project folder
cd knn-dashboard

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
